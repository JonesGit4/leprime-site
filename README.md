# Le Prime Acabamentos - Website

Site institucional da Le Prime Acabamentos.

**URL:** https://www.leprimeacabamentos.com.br

## Deploy

Este projeto é um site estático hospedado no Vercel, conectado a este repositório GitHub.

Qualquer push para a branch `main` faz deploy automático.

## Estrutura

```
public/           # Arquivos estáticos do site
├── index.html    # Página principal
├── images/       # Imagens e logos
├── _next/        # Assets estáticos (CSS, JS, fontes)
└── *.jpg         # Imagens de produtos
vercel.json       # Configuração do Vercel
```
